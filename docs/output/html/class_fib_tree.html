Detailed Description

template<typename T>
class FibTree< T >

Fibonacci Heap data structure implementation.

This class implements a Fibonacci heap, a priority queue data structure that supports efficient insertion, extraction of minimum key, and key decrease operations.

Template Parameters
TType of elements to be stored in the heap.
